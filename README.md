# REA Toy Robot

## Technology details

### Stack
- Node.js
- Mocha
- Chai
- Travis (CI)
- Coveralls

---

### How to Run

#### Pre Reqs
- NPM
- YARN
##### *Note* Use as per choice

#### Install Dependencies
```yarn```

#### Run Solution (Interactive)
```yarn toy:run```

#### Test Solution
```yarn toy:test```

#### Code Coverage (Coveralls)
```yarn toy:code_coverage```
##### *Note* See travis after success step for more info
